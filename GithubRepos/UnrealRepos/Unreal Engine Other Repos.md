## Abstractions
[J-Andersen-UvA/UnrealAssetFileManagement](https://github.com/J-Andersen-UvA/UnrealAssetFileManagement)
Editor Abstraction to manage files outside of unreal engine.

[J-Andersen-UvA/sequencerAbstraction](https://github.com/J-Andersen-UvA/sequencerAbstraction)
Editor Abstraction to manage the sequencer. You can add stuff to the sequencer, see whats inside it, open the window, and move timing.
## Hardware
[J-Andersen-UvA/HueControl](https://github.com/J-Andersen-UvA/HueControl)
Plugin that adds the ability to control Philips HUE system within Unreal Engine.

[J-Andersen-UvA/UnrealMidi](https://github.com/J-Andersen-UvA/UnrealMidi)
Plugin that adds support of MiDi based controllers to Unreal Engine. Made for editor, but probably not hard to translate to runtime (might even already work there).

[J-Andersen-UvA/AzureKinectUEPlugin](https://github.com/J-Andersen-UvA/AzureKinectUEPlugin)
Plugin that adds the ability to read and work with the Azure Kinect library. You will need the Body Tracking SDK for this to work.
