## AutoRetargetUnrealHeadless
[J-Andersen-UvA/AutoRetargetUnrealHeadless](https://github.com/J-Andersen-UvA/AutoRetargetUnrealHeadless)
An implementation of Unreal's retargeting system to use a retarget asset by means http requests. Useful for automation of retargeting large numbers of animations.

## RetargetAbstraction
[J-Andersen-UvA/UERetargetModule: Retarget wrapper, and more about retargeting in Unreal!](https://github.com/J-Andersen-UvA/UERetargetModule)
Just some wrappers for retargeting in Unreal Engine through cpp, bp, or python.
