1. Define folder struct.
	"main" with 2 subfolders: "in" & "out".
2. Unreal import a folder of animations onto an avatar with correct settings
	Use fileManaging for import: [J-Andersen-UvA/UnrealAssetFileManagement](https://github.com/J-Andersen-UvA/UnrealAssetFileManagement)
3. Retarget all from folder to another folder.
	Use RetargetAbstraction for retarget: [J-Andersen-UvA/UERetargetModule: Retarget wrapper, and more about retargeting in Unreal!](https://github.com/J-Andersen-UvA/UERetargetModule)
4. Export it out of unreal to the out folder.
	Use fileManaging for export: [J-Andersen-UvA/UnrealAssetFileManagement](https://github.com/J-Andersen-UvA/UnrealAssetFileManagement)
